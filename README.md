# Vue 3 + Typescript + Vite

### 开发

`yarn dev`

### 打包

`yarn build`

### 部署到 http://www.hosea.shop

`yarn dis:www`

### 部署到 http://test.hosea.shop

`yarn dis:test`

###### http://www.hosea.shop

###### 后台页面：http://www.hosea.shop/management

#### 测试地址：
